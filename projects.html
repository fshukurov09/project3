Hardware Logic Lab

SITE 1101 | Fall 2025

An in-depth exploration of digital logic fundamentals through hands-on hardware implementation. This project involved constructing basic logic gates (AND, OR, NOT, XOR) using transistors and other electronic components, demonstrating how Boolean algebra translates into physical circuits.

Key Components:

  • AND Gate: Implemented using transistor configuration to output true only when all inputs are true
  • OR Gate: Built circuit that outputs true when any input is true
  • NOT Gate: Created inverter circuit using transistor switching
  • XOR Gate: Constructed exclusive-or gate combining multiple logic operations

Learning Outcomes:

  • Understanding transistor-level operation and switching behavior
  • Connecting theoretical Boolean logic to physical hardware implementation
  • Debugging circuit designs and troubleshooting signal issues
  • Appreciation for how modern processors build complex operations from simple gates
